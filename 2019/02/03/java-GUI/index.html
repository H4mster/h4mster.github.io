GUI

GUI，即Graphic User Interface 图形用户界面

Hello Swing

step1. 简单的例子

JFrame是GUI中的容器
JButton是最常见的组件-按钮
注: f.setVisible(true); 会对所有组件进行渲染，所以一定要放在最后面

step2. 在上次关闭的位置启动窗口

启动一个线程，每个100毫秒读取当前的位置信息，保存在文件中，比如location.txt文件。
启动的时候，从这个文件中读取位置信息，如果是空的，就使用默认位置，如果不是空的，就把位置信息设置在窗口上。
读取位置信息的办法： f.getX() 读取横坐标信息，f.getY()读取纵坐标信息。

时间监听

关键字 简介
ActionListener 按钮监听
KeyListener 键盘监听
MouseListener 鼠标监听
Adapter 适配器
step1. 按钮监听

创建一个匿名类实现 ActionListener 接口，当按钮被点击时， actionPerformed 方法就会被调用

step2. 键盘监听

键盘监听器: KeyListener

keyPressed 代表 键被按下

keyReleased 代表 键被弹

keyTyped 代表 一个按下弹起的组合动作

KeyEvent.getKeyCode() 可以获取当前按下了哪个键

step3. 鼠标监听

MouseListener 鼠标监听器

mouseReleased 鼠标释放

mousePressed 鼠标按下

mouseExited 鼠标退出

mouseEntered 鼠标进入

mouseClicked 鼠标点击
在本例中，使用mouseEntered，当鼠标进入图片的时候，图片就移动位置

step4. 适配器

一般来说写监听器的时候，会实现上述的三种接口，但是实际上很多方法都没有用到，这时候用适配器就行了，适配器是一个抽象类，继承了监听器的接口。

Sr.NO. 适配器 说明
1 FocusAdapter 接收焦点事件的抽象适配器类
2 KeyAdapter 接收按键事件的抽象适配器类
3 MouseAdapter 接收鼠标事件的抽象适配器类
4 MouseMotionAdapter 接收鼠标移动事件的抽象适配器类
5 WindowAdapter 接收窗口事件的抽象适配器类

容器

java的图形界面中，容器是用来存放 按钮，输入框等组件的。
窗体型容器有两个，一个是JFrame,一个是JDialog

step1. JFrame

JFrame是最常用的窗体型容器，默认情况下，在右上角有最大化和最小化按钮。

step2. JDialog

JDialog也是窗体型容器，右上角没有最大和最小化按钮

step3. 模态JDialog

当一个对话框被设置为模态时，其背后的父窗体是不能被激活的，除非该对话框被关闭

step4. 窗体大小不可变化

通过调用方法 setResizable(false); 做到窗体大小不可变化

step5. 练习-模态与大小变化

首先设计一个JFrame,上面有一个按钮，文字是 “打开一个模态窗口”。

点击该按钮后，随即打开一个模态窗口。

在这个模态窗口中有一个按钮，文本是 “锁定大小”, 点击后，这个模态窗口的大小就被锁定住，不能改变。 再次点击，就能够改变大小。
